Guide de Téléchargement

Comment télécharger les fichiers partagés entre médecins

1Ouvrir une conversation

Sélectionnez une conversation avec un médecin dans la liste de gauche.

2Trouver un fichier

Identifiez un message contenant des fichiers attachés (affichés sous le texte du message).

Voici le rapport que vous m'avez demandé

rapport_medical.pdf
2.4 MB

3Cliquer sur le bouton de téléchargement

Cliquez sur l'icône de téléchargement à droite du fichier.

4Le fichier est téléchargé

Le fichier sera automatiquement téléchargé dans votre dossier "Téléchargements".

Succès!

Une notification "Fichier téléchargé: rapport_medical.pdf" s'affichera.

Fonctionnalités

Prévisualisation

Les images peuvent être prévisualisées avant téléchargement

Tous les formats

PDF, Word, Excel, Images, Archives, etc.

Sécurisé

Accès restreint aux médecins de la conversation

Rapide

Téléchargement instantané jusqu'à 50 MB

Astuce:

Pour les images, cliquez directement sur le fichier pour le prévisualiser dans une modal. Vous pourrez ensuite le télécharger depuis la modal si vous le souhaitez.

Sur Mobile:

Le téléchargement fonctionne de la même manière. Les fichiers seront enregistrés dans votre gestionnaire de téléchargements.
